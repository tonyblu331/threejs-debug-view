`threejs-debug-view` gives you named debug views for the buffers your scene already produces: beauty, normals, depth, material channels, wireframe, lighting-only, reflection-only, overlap, and shader cost.

Use the root package for core helpers and TSL view definitions — no React or R3F required. Use `threejs-debug-view/r3f` when you want the batteries-included overlay inside a React Three Fiber `<Canvas>`.

## Batteries included

The `/r3f` entrypoint is the fast path:

```tsx
import { DebugViewLayer } from "threejs-debug-view/r3f"

<DebugViewLayer />
```

That one component mounts the overlay, wires Leva controls on `document.body` (outside the R3F tree), and registers the built-in debug sources — no separate `<Leva />` import. See [Batteries Included](/threejs-debug-view/guides/batteries-included/) for what ships out of the box, what stays optional, and what never lands on npm.

## R3F adapter

The R3F adapter is the React UI layer. `DebugViewLayer` mounts the default overlay inside your canvas, reads the active R3F renderer, and portals the bundled Leva panel to `document.body`. Use `DebugViews` and `useDebugViewsControls` when your app needs to own control state instead.

Keep it behind your app's dev flag unless you intentionally publish a debug surface.

## Live demo

Use the controls in the top-right panel to switch views, layouts, opacity, and pane count. On the **Shader Complexity** view, click the viewport to sample a pixel and align the legend marker with what you clicked.

## What it does

- **Built-in sources** for beauty, normals, depth, albedo, roughness, AO, opacity, emissive, wireframe, lighting-only, reflection-only, overlap, and shader cost.
- **Render layouts** for single, overlay, split, diagonal breakdown, row, column, and grid presentations.
- **Explicit pane assignments** for fixed views, labels, per-pane cameras, and predictable resolution scaling.
- **Reusable runtime pipelines** for common material-channel switching without material mutation.
- **Custom TSL views** for shader-specific inspection without forking the built-in source list.
- **R3F adapter** for mounting the overlay and optional Leva controls inside a React Three Fiber app.
